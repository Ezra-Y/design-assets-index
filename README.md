## Why

When you ask an agent for "a nice placeholder image" or "an icon set for a sidebar", how-to design skills tell it the *principles* вҖ” but not *which site to grab from*. This skill fills that gap: a navigable catalog of trustworthy asset sources, distilled from the [awesome-design](https://github.com/gztchan/awesome-design) and [design-resources-for-developers](https://github.com/bradtraversy/design-resources-for-developers) awesome-lists.

> рҹ’Ў **Type note:** This is a *resource index* skill (links + one-line notes). It is deliberately not a how-to/teaching skill.
