# Design Assets Index

外部设计素材导航，告诉 agent「去哪找」现成素材。每个条目为「名称（链接）— 一句话说明」，精选自 [awesome-design](https://github.com/gztchan/awesome-design) 与 [design-resources-for-developers](https://github.com/bradtraversy/design-resources-for-developers)。更多条目见 [references/more.md](references/more.md)。

## 图库素材

- [Unsplash](https://unsplash.com/) — Free high-resolution photos, no attribution required
- [Pexels](https://www.pexels.com/) — Free stock photos & videos shared by creators
- [Pixabay](https://pixabay.com/) — Over 1M+ free stock images and videos
- [The Stocks](http://thestocks.im/) — Aggregates free photos, videos, icons, fonts and colors
- [AllTheFreeStock](http://allthefreestock.com/) — Index of free stock photos, videos, icons and music
- [UI Faces](https://uifaces.co/) — Aggregated avatars for personas and placeholders
- [Gratisography](http://www.gratisography.com/) — Quirky free high-resolution stock images
- [FoodiesFeed](https://www.foodiesfeed.com/) — Free natural-looking food photos

## 图标

- [Tabler Icons](https://tabler-icons.io/) — 3500+ customizable open-source SVG icons
- [Lucide](https://lucide.netlify.app/) — Open-source icon library, fork of Feather Icons
- [Iconify](https://iconify.design/) — One framework, 40,000+ vector icons unified
- [Phosphor Icons](https://phosphoricons.com) — Flexible icon family for interfaces
- [Heroicons](https://heroicons.dev/) — Free open-source icons by the Tailwind creators
- [Simple Icons](https://simpleicons.org/) — Free SVG icons for popular brands
- [Material Design Icons](https://materialdesignicons.com/) — Icons in any format, size or color
- [The Noun Project](https://thenounproject.com/) — 2M+ curated icons by a global community

## 字体

- [Google Fonts](https://fonts.google.com/) — Library of ~1000 free licensed font families
- [Font Squirrel](https://www.fontsquirrel.com/) — 100% free for commercial use
- [Fontshare](https://www.fontshare.com/) — Free quality fonts from the Indian Type Foundry
- [Fontsource](https://fontsource.org/) — Self-host open-source fonts via NPM packages
- [DaFont](https://www.dafont.com/) — Archive of freely downloadable fonts
- [Open Foundry](https://open-foundry.com) — Curated open-source typefaces
- [Typewolf](https://www.typewolf.com/) — What's trending in type
- [The League of Moveable Type](https://www.theleagueofmoveabletype.com/) — Open-source type foundry

## 配色

- [Coolors](https://coolors.co) — Super-fast color scheme generator
- [Adobe Color](https://color.adobe.com/create) — Build palettes, extract gradients from images
- [Color Hunt](https://colorhunt.co/) — Thousands of hand-picked palettes
- [Color Brewer 2](https://colorbrewer2.org/) — Palette generator with color-blindness support
- [UI Colors](https://uicolors.app/create) — Tailwind CSS color palette generator
- [Brand Colors](https://brandcolors.net/) — Official brand color codes collection
- [WebGradients](https://webgradients.com/) — 180 free linear gradients for backdrops
- [Happy Hues](https://www.happyhues.co/) — Palette inspiration with real-world usage

## Mockup

- [Mockup World](https://www.mockupworld.co/) — Largest source of free photo-realistic mockups
- [Smart Mockups](https://smartmockups.com/) — Create product mockups (free & premium)
- [Shotsnapp](https://shotsnapp.com/) — Device mockup images for app/website designs
- [Screely](https://www.screely.com/) — Turn a screenshot into a mockup instantly
- [Screenshot.rocks](https://screenshot.rocks/) — Browser & mobile mockups in seconds
- [Facebook Devices](https://facebook.design/devices) — Sketch files of popular devices
- [Threed.io](https://threed.io) — Generate 3D mockups in the browser
- [Mockuphone](https://mockuphone.com/) — 100% free mockups for all devices

## UI Kit

- [UI Bundle](https://uibundle.com) — Free UI kits, mockups, illustrations, icons, fonts, 3D
- [UI Design Daily](https://uidesigndaily.com/) — UI components of all types
- [undraw](https://undraw.co/) — Open-source illustrations for any idea
- [Humaaans](https://www.humaaans.com/) — Mix-and-match illustrations of people
- [Blush](https://blush.design/) — Customizable illustrations with Figma plugin
- [uiverse](https://uiverse.io) — Hundreds of open-source UI elements (HTML & CSS)
- [Headless UI](https://headlessui.dev/) — Unstyled, accessible components for Tailwind
- [Creative Tim](https://www.creative-tim.com/) — UI libraries and kits (React, Vue, etc.)

## 模板

- [HTML5 Up](https://html5up.net/) — Modern, unique responsive HTML5/CSS3 themes
- [Start Bootstrap](https://startbootstrap.com/) — Bootstrap starter themes
- [Colorlib](https://colorlib.com/wp/templates/) — Themes in almost any category
- [Cruip](https://cruip.com/free-templates/) — Coded HTML templates for landing pages
- [Templatemo](https://templatemo.com/) — Minimal, resume, gallery themes and more
- [StyleShout](https://www.styleshout.com/free-templates/) — Crafted free website templates
- [Bootswatch](https://bootswatch.com/) — Free themes for Bootstrap
- [Onepagelove](https://onepagelove.com/) — One-page websites, templates and resources
